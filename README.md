## Getting Started

install deps::

```bash
npm install

# or

yarn
```

add `.env.local` at the root and populate with secrets

```
NEXT_PUBLIC_G4TAG=**********
NEXT_PUBLIC_SANITY_PROJECT_ID=**********
NEXT_PUBLIC_SANITY_DATASET=**********
SANITY_API_READ_TOKEN=**********

```

run the dev server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser
